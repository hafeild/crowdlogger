Implementing a CrowdLogger Remote Module

This page will contain information about implementing a module, including the directory structure and some simple examples. It will also contain an overview of the API.

What a CLRM consists of

A CLRM is a JSON object with the following fields:

Field Type Description
module string The core module code. This must adhere to the structure described below.
html map to strings An map of HTML document names to the document content. Each of these virtual files can be loaded into a window when needed. They can reference other sources from this object, including JavaScript (see js), CSS (see css), and other HTML content. See the Referencing module resources section for more information about how to link to these resources.
js map to strings The core module code. This must adhere to the structure described below.
css map to strings The core module code. This must adhere to the structure described below.

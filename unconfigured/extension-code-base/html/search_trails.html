This is a tool similar to your browser's web history tool. However, this is different in that we make a distinction between three levels of activities: searches, search result clicks, and non-search result clicks. We also do not currently display pages that were not visited via a click (that will change in future releases).

This is a first pass on what will hopefully become a very useful tool to explore your search and browsing history. We plan to add additional functionality, such as task identification. If you have suggestions for things to add to the search trails tool or for other %%PROJECT_NAME%% tools, please email %%CONTACT_EMAIL%%.

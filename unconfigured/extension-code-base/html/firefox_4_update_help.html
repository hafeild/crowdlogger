Firefox Updating Instructions
for
%%PROJECT_NAME%%
To update %%PROJECT_NAME%% for Firefox, go to the Add-ons manager by clicking here. On the left side of the page, select Extensions. In the upper right corner of that page, you will see a search box and to the left of that you will see an icon of a gear. Click that gear. In the drop down menu, select Check for Updates. If there are updates, Firefox will let you know and you can select to install them.

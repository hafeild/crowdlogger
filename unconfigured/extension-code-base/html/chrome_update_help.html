Google Chrome Updating Instructions
for
%%FULL_PROJECT_NAME%%
To update %%PROJECT_NAME%% in Google Chrome, go to the Chrome tools menu (click on the wrench icon next to the web address bar). Select Tools → Extensions. This will bring you to the extensions page. In the upper right part of the page you should see something that says Developer mode. Click that. That will expose several buttons to the left, one of which is labeled Update extensions now. Click that button to update all of your extensions, including %%PROJECT_NAME%%.

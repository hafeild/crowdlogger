Firefox 3 Updating Instructions
for
%%PROJECT_NAME%%
To update to the newest version of %%PROJECT_NAME%% in Firefox 3, go to Tools → Add-ons. That will open the Add-ons dialog. Click on the Extensions tab. At the bottom of the dialog, you will see a button labeled Find Updates. Click that. Firefox will then show you what add-ons have updates available and you can select to install them.

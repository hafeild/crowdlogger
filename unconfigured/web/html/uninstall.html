Uninstalling %%PROJECT_NAME%% for Google Chrome

To uninstall %%PROJECT_NAME%% for Chrome, right click the %%PROJECT_NAME%% icon on the navigation bar. This will bring up a Chrome menu. Click on the "Remove from Chrome" option to uninstall the extension. This will remove all data associated with the extension from your computer. Note that you can also disable the extension, which will shut the extension down and remove the icon from the navigation bar, but all stored data will remain on your computer. You can re-enable an extension by clicking the Chrome options menu (far right button in the navigation bar), selecting "Tools" then "Extensions". Locate %%PROJECT_NAME%% (it will be grayed out) and check the box next to "Enable".

Uninstalling %%PROJECT_NAME%% for Firefox

Perform the following steps:

1. Open the Add-ons page—this is done differently based on operating system:

1a. If you are using Windows 7 or above, click the "Firefox" menu in the upper left and select "Add-ons"

1b. Otherwise, you should have a "Tools" menu; go to "Tools"->"Add-ons".

1c. You can also enter the URL about:addons.

2. A new tab will be displayed. Select "Extensions" on the left.

3. Find and select the "%%PROJECT_NAME%%" in the list of installed extensions. Click on the "Remove" button and Firefox will uninstall the extension.

4. You will be asked if you would like to remove all data associated with %%PROJECT_NAME%%, such as search history, app data, and settings. Choose "OK" if you would like to remove this data. Choosing "Cancel" will cause %%PROJECT_NAME%% to be removed, but the data will remain; if you reinstall %%PROJECT_NAME%% later, it will have access to your old data and preferences.
